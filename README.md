# Redstone Volume

A browser extension to control volume in a redstone-inspired way.

## 🚀 Installation

1. Download or clone the repository:

```bash
git clone https://github.com/as1hkan/Redstone-Volume.git
```

2. Open your browser and go to extensions page:

* Chrome: `chrome://extensions/`
* Edge: `edge://extensions/`

3. Enable **Developer mode** (top right corner)

4. Click on **Load unpacked**

5. Select the project folder (`Redstone-Volume`)

6. Done — the extension is now installed

## ▶️ Usage

* Click on the extension icon
* Use it to control volume
